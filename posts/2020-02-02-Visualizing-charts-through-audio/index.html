Visualizing charts through audio

by Maximilian Götzfried on February 2, 2020

Audiograph by Andreas Neusüß is a great Swift framework that visualizes charts through audio for visually impaired people. A new feature of iOS 13 that is not available as a public API yet.

Check the videos in the README or this tweet to see in action.
